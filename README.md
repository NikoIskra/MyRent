# Object browser

Web aplikacija koja koristi dostupne API metode i prikazuje podatke o objektima

## Lokalno pokretanje

Potrebno je dodati vrijednosti za token i guid u MyRent/appsettings.json

```bash
"MyRentSettings": {
    "BaseUrl": "https://apit.my-rent.net/",
    "Guid": "xxxxx",
    "Token": "yyyyy"
  },
```
